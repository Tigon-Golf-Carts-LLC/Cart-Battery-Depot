## WCAG ACCESSIBILITY COMPLIANCE

### WCAG 2.1 Level AA Compliance

Cart Battery Depot is committed to web accessibility and strives to meet
WCAG 2.1 Level AA guidelines across all pages and features.

### Accessibility Features

#### Perceivable
- Alt text for all meaningful images
- Captions available for video content
- Sufficient color contrast (4.5:1 minimum)
- Content adaptable to different presentations
- Text resizable up to 200%

#### Operable
- Keyboard accessible navigation
- No keyboard traps
- Skip navigation links
- Focus indicators visible
- Sufficient time for interactions
- No content that causes seizures

#### Understandable
- Consistent navigation
- Predictable functionality
- Input assistance and error messages
- Clear language (en-US)
- Error prevention for important actions

#### Robust
- Valid HTML markup
- ARIA landmarks and roles
- Compatible with assistive technologies
- Progressive enhancement approach

### Assistive Technology Support
- Screen readers (JAWS, NVDA, VoiceOver)
- Voice control software
- Switch devices
- Screen magnification
- High contrast modes
